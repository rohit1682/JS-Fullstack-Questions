Dynamo db is a no sql db which stores data using key value pair and is serverless.
It is called serverless because we dont need to manage the instance of any db. we dont need to create reader or writer instance.
Dynamo db has a drawback that if the db has 1gb of data but it needs to scan 1kb of data, it will run for the entire storage which increases the cost and latency.
partition keys work as primary key and indexes work as secondary keys.

s3 is used in auditing, ott platforms, static content websites
