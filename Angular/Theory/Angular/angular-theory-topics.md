# Angular Theory Topics

This folder contains topic-wise Angular explanations designed for clear learning and interview preparation.

## Files

- `components.md` — Angular component basics and communication patterns
- `templates-data-binding.md` — template syntax, bindings, and best practices
- `directives.md` — built-in and custom directives
- `services-dependency-injection.md` — services, providers, and DI scope
- `routing.md` — router setup, route parameters, guards, and lazy loading
- `lifecycle-hooks.md` — component lifecycle hooks and usage
- `pipes.md` — built-in pipes, custom pipes, and performance
- `NgModule.md` — module anatomy, imports/exports, and feature modules
- `standalone-components.md` — standalone component architecture and bootstrapping
- `forms.md` — template-driven and reactive forms with validations
- `change-detection.md` — change detection strategies and manual control

## Existing reference files

- `Basics.md` — overview of Angular concepts
- `NgModuleVSStandalone.md` — compare NgModule and standalone approaches
- `Theory.md` — general Angular overview
