# Angular

Angular is a TypeScript-based, open-source framework by Google used to build scalable Single Page Applications (SPAs).

# Core Concepts

1. Component-Based Architecture → Apps are built using reusable components (logic + UI + styles).
2. TypeScript → Strong typing improves code quality and tooling.
3. Templates & Data Binding → Easy UI updates using:

{{ }} (interpolation)
[ ] (property binding)
( ) (event binding)
[( )] (two-way binding)

# Key Features

1. Dependency Injection (DI) → Clean, testable, and modular code.
2. Built-in Testing → Tools like Jasmine, Karma, TestBed.
3. First-party libraries → Routing, forms, HTTP, animations, PWA support.

# Tooling & Performance

1. Angular CLI → Fast project setup and development.
2. AOT Compilation → Better performance.
3. Standalone Components → Simpler structure (no NgModules required).

# Modern Updates (Angular 16+)

1. Signals → New reactivity system
2. New control flow → @if, @for
3. Better SSR & performance improvements
